JAS REST API Documentation

Recording temperature data

POST /API/SetTemperatureRecord/$device_id$

[ { 'sensorID' : $sensor_id$, 'temperature': $data$, 'timestamp' : $datetime$ }, ]
Array of :
  • $device_id$:string value of the identity number of an active device
  • $sensorid$:string, value of the identity number ot the sensor
  • $data$:float, the measured temperature
  • $datetime$:datetime, the date and time of the measure. Automaticly added if blank.
